لجعل الرفع يعمل لجميع الزوار

1

إنشاء مشروع Firebase

اذهب إلى console.firebase.google.com وأنشئ مشروعاً جديداً باسم "dar-aishe"

2

إضافة تطبيق ويب

في المشروع، اضغط على أيقونة الويب (</>) وسجّل التطبيق، ثم انسخ بيانات الإعداد

3

تفعيل Firestore و Storage

من القائمة الجانبية: Build → Firestore Database → Create (test mode)
ثم: Build → Storage → Get started (test mode)

4

الصق بيانات الإعداد

افتح ملف firebase-config.js والصق القيم من Firebase Console
